# Blockchain-Hackathon

これはBCHのリポジトリです。

## システム

・ **Vite を使用**

<pre>
$ npm create vite@latest
$ cd BCH
$ npm install
</pre>

・ **実行方法**

<pre>
$ cd BCH
$ npm run dev
</pre>

・ CSSには **TailwindCSS** を使用

・ Iconには **React-Icons** を使用
